1. Python3 standard type
    1) None
        - NoneType
   
    2) Numbers
        - int(정수), float(실수), boolean

    3) Sequences
        - Mutable 
            - list (자유롭게 변경이 가능한 여러 개의 데이터 묶음)

            - tuple (생성 후에 변경 불가능한 여러 개의 데이터 묶음)
            - str (문자열,character sequence)

    4) set
        - set (집합)

    5) Mappings
        - dict (key-value store 방식을 가지고 자료를 저장하는 구조)
